Язык C# и платформа .NET

Коротко о C#

  • Является объектно-ориентированным (поддерживает полиморфизм, наследование, абстракцию, статическую типизацию);
  • Первая версия языка вышла в феврале 2002 года (вместе с релизом Microsoft Visual Studio .NET);
  • Текущая версия: C# 12 (вместе с .NET 8) вышла 14 ноября 2023 года;
  • Реализует JIT-компиляцию (Just-In-Time, в текущий момент времени компилируется лишь та часть приложения/кода, к которой идет обращение. Скомпилированные ранее части кода также хранятся в памяти до завершения работы приложения, что повышает производительность).

Приложения, написанные на C#, часто называют управляемым кодом (managed code), т.к. приложение управляется общеязыковой средой CLR, которая берет на себя все взаимодействия с системой, на которой выполняется приложение (управление ресурсами, памятью, процессорным временем и пр.). Однако C# предоставляет инструменты для взаимодействия с неуправляемым кодом (когда код приложения компилируется в обычный машинный код).

Коротко о платформе .NET

Платформа .NET - мощная платформа для создания приложений, которая обладает следующими свойствами:

  • Поддерживает несколько языков: В основе платформы лежит общеязыковая среди исполнения Common Language Runtime (CLR), благодаря чему .NET поддерживает несколько языков (C#, VB.NET, C++, F# и пр.). При компиляции код компилируется в сборку на общем языке Common Intermediate Language (CIL) - своего рода ассемблер платформы;
  • Кроссплатформенность: .NET является переносимой платформой (с некоторыми ограничениями) и поддерживается на большинстве современных ОС Windows, MacOS и Linux;
  • Мощная библиотека классов: .NET предоставляет единую для всех поддерживаемых языков библиотеку классов;
  • Разнообразие технологий: Для платформы написано очень много различных фреймворков, начиная с ADO.NET и Entity Framework Core и заканчивая ASP.NET и Blazor;
  • Производительность: Согласно ряду тестов веб-приложения на .NET в ряде категорий сильно опережают веб-приложения, построенные с помощью других технологий.

Ранее платформа .NET развивалась как платформа преимущественно под Windows и называлась .NET Framework, однако с 2014-го года началось развитие альтернативной платформы, названной .NET Core. Последняя версия .NET Framework (4.8) вышла в 2019-ом году.

Синтаксис C# в двух словах

  1. Общие соображения:

    • Регистрозависимый язык (name, Name и NAME - разные переменные);
    • Код программы состоит из инструкций (statement). Инструкцией может быть объявление переменной, вызов функции, арифметическая операция и пр. После каждой инструкции должна ставиться точка с запятой*;

  2. Переменные

    • Имя переменной может содержать любые цифры, буквы и символ подчеркивания (но первый символ не может быть цифрой);
    • Имя переменной не может быть названо ключевым словом языка C#.

  3. Константы

    • Имеют такие же ограничения по наименованию, что и переменные;
    • Должны быть инициализированны до компиляции.

  4. Литералы

    • null
    • Логические (true, false);
    • Целочисленные (1, 2, 3, -5, 0, 42);
    • Вещественные (3.14, 100.23, -2.71828);
    • Символьные ('A', 'b', 'd');
    • Управляющие последовательности ('\n', '\t', '\\');
      Могут быть представлены в виде:
      • Коды ASCII ('\x78', '\x30')
      • Коды Unicode ('\u0420', '\u4139')
    • Строковые ("test", "help", "i need more mana").
